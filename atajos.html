Terminal básica

Comandos esenciales que sí vale la pena conocer.

La terminal no es obligatoria para todo, pero entender estos comandos te da más control y seguridad.

pwd

Ver dónde estás

Muestra la carpeta actual.

ls

Listar archivos

Muestra el contenido de una carpeta.

cd Documentos

Entrar a una carpeta

Cambia de ubicación dentro del sistema.

mkdir proyecto

Crear carpeta

Crea una nueva carpeta llamada proyecto.

cp archivo.txt copia.txt

Copiar archivo

Crea una copia de un archivo.

mv viejo.txt nuevo.txt

Mover o renombrar

Sirve para mover archivos o cambiar nombres.

rm archivo.txt

Eliminar archivo

Úsalo con cuidado. No siempre hay papelera.

sudo comando

Ejecutar como administrador

Solo úsalo si sabes qué hace el comando.

Gestión de paquetes

Actualizar e instalar software según tu distribución.

Cada familia de Linux usa comandos distintos. Altépet siempre te deja una guía específica según la distribución que instalemos.

Ubuntu / Linux Mint / Zorin

sudo apt update

sudo apt install nombre

Fedora / Nobara / Bazzite

sudo dnf update
sudo dnf install nombre

Arch / Garuda / CachyOS

sudo pacman -Syu

openSUSE

sudo zypper refresh

sudo zypper install nombre

Preguntas frecuentes

Antes de usar la terminal.

¿Tengo que usar terminal para usar Linux?

No. Muchas tareas se hacen con interfaz gráfica. La terminal solo te da más control cuando la necesitas.

¿Es peligroso copiar comandos de internet?

Sí puede serlo. Nunca ejecutes comandos que no entiendas, especialmente si incluyen sudo o rm -rf.

¿Qué significa sudo?

Ejecuta comandos con permisos de administrador. Úsalo con cuidado porque puede modificar partes importantes del sistema.

¿Altépet me deja una guía personalizada?

Sí. La idea es que no dependas de memorizar todo, sino de tener una ruta clara según tu distribución.
