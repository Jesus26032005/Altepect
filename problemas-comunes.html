Diagnóstico rápido

Antes de ejecutar comandos: observa esto.

1

¿Cuándo empezó?

Después de instalar, actualizar, conectar un dispositivo o cambiar configuración.

2

¿Afecta a todo o solo a una app?

Si solo falla una app, probablemente no es problema del sistema completo.

3

¿Tu hardware está detectado?

WiFi, audio, Bluetooth e impresoras dependen mucho del hardware y drivers.

4

¿Hay espacio y memoria?

Un disco lleno o poca RAM puede hacer que todo parezca roto.

Casos frecuentes

Problemas comunes y rutas seguras.

Estas tarjetas no reemplazan soporte técnico, pero te ayudan a entender qué revisar primero.

📶

Red

El WiFi no aparece

Puede deberse a drivers, firmware faltante, modo avión o adaptadores no soportados.

Revisa primero

  • Que el modo avión esté desactivado.
  • Si Ethernet funciona.
  • Si el adaptador aparece en configuración de red.
ip a

Altépet revisa chipset WiFi antes de recomendar una distribución.

🌐

Internet

Tengo red, pero no cargan páginas

Si estás conectado pero no navegas, puede ser DNS, router o configuración de red.

Revisa primero

  • Prueba otra página.
  • Reinicia router.
  • Comprueba si otros dispositivos navegan.
ping 1.1.1.1

Si responde por IP pero no por nombre, probablemente es DNS.

🔊

Audio

No tengo sonido

Puede ser salida incorrecta, volumen silenciado, PipeWire/PulseAudio o hardware no detectado.

Revisa primero

  • Salida de audio seleccionada.
  • Volumen del sistema y de la app.
  • Audífonos Bluetooth o HDMI activos por error.
pactl info

Altépet deja configuradas salidas comunes y prueba audio antes de entregar.

🟦

Bluetooth

Bluetooth no conecta

Los problemas más comunes son servicio detenido, emparejamiento previo o adaptador no reconocido.

Revisa primero

  • Olvida el dispositivo y empareja de nuevo.
  • Verifica batería del dispositivo.
  • Reinicia Bluetooth desde ajustes.
systemctl status bluetooth

En laptops antiguas a veces conviene usar adaptador Bluetooth USB compatible.

🖨️

Hardware

No aparece la impresora

Linux soporta muchas impresoras, pero algunas requieren drivers, CUPS o configuración de red.

Revisa primero

  • Si la impresora está en la misma red.
  • Si imprime desde otro equipo.
  • Marca y modelo exacto.
systemctl status cups

Altépet puede revisar compatibilidad antes de migrar equipos de oficina.

🐢

Rendimiento

Linux va lento

Puede ser falta de RAM, disco duro mecánico, entorno pesado, apps al inicio o poco espacio.

Revisa primero

  • Uso de RAM y CPU.
  • Espacio disponible.
  • Si usas GNOME/KDE en hardware muy limitado.
free -h && df -h

Muchas laptops mejoran mucho con SSD + XFCE/Mint/Lubuntu.

💾

Almacenamiento

El disco está lleno

Un disco lleno causa lentitud, errores de actualización y fallos al abrir aplicaciones.

Revisa primero

  • Descargas y videos pesados.
  • Cachés antiguas.
  • Snapshots o backups acumulados.
df -h

No borres carpetas del sistema sin saber qué hacen.

🎮

Gráficos

Problemas con NVIDIA

NVIDIA puede requerir drivers propietarios, versión correcta del kernel y configuración especial.

Revisa primero

  • Distribución usada.
  • Modelo exacto de GPU.
  • Si falla después de actualizar.
lspci | grep -i nvidia

Para gaming, Altépet elige distros pensando en tu GPU.

📄

Software

No encuentro Word o Excel

En Linux puedes usar Microsoft 365 en navegador, Google Docs o LibreOffice.

Revisa primero

  • Si necesitas macros avanzadas.
  • Compatibilidad con documentos de escuela/trabajo.
  • Uso en navegador vs app local.
libreoffice

Altépet configura la opción más práctica según tu caso.

🕹️

Gaming

Steam o juegos no abren

Puede ser Proton, drivers gráficos, librerías faltantes o compatibilidad del juego.

Revisa primero

  • Activar Steam Play/Proton.
  • Actualizar drivers.
  • Consultar compatibilidad del juego.
steam

Para gaming recomendamos Bazzite, Nobara, Garuda o Pop!_OS según hardware.

🧩

Aplicaciones

No encuentro mi programa

Algunos programas no tienen versión Linux, pero puede haber alternativas, web apps o ejecución por compatibilidad.

Revisa primero

  • Si existe versión web.
  • Alternativas libres.
  • Si necesitas Windows para un programa crítico.
flatpak search nombre

Antes de migrar, Altépet revisa tus programas indispensables.

🔄

Actualizaciones

Fallan las actualizaciones

Puede deberse a repositorios caídos, paquetes bloqueados, conexión o una actualización interrumpida.

Revisa primero

  • Conexión estable.
  • No cerrar el equipo durante actualizaciones.
  • Leer el mensaje de error completo.
sudo apt update

Si no sabes qué significa el error, no fuerces comandos destructivos.

Buenas prácticas

Qué NO hacer cuando algo falla.

No pegues comandos peligrosos

Evita comandos con rm -rf, permisos de administrador o scripts externos si no sabes qué hacen.

No borres carpetas del sistema

Carpetas como /etc, /usr o /boot son delicadas.

No reinstales sin respaldo

Antes de cualquier cambio grande, respalda documentos, fotos y archivos de trabajo.

No mezcles tutoriales al azar

Un tutorial para Arch no siempre aplica a Ubuntu. Uno para Fedora no siempre aplica a Mint.

FAQ de soporte

Dudas comunes cuando algo se rompe.

¿Tengo que reinstalar Linux si algo falla?

No necesariamente. Muchos problemas se resuelven ajustando drivers, servicios, repositorios o configuración.

¿Linux falla menos que Windows?

Linux puede ser muy estable, pero depende de hardware, distribución, drivers y configuración. Un buen diagnóstico importa mucho.

¿Altépet puede revisar mi equipo antes de instalar?

Sí. Esa es una parte clave del servicio: revisar hardware, programas críticos y necesidades reales.

¿Puedo pedir soporte después de la instalación?

Sí. Altépet está pensado como servicio con acompañamiento, no solo como instalación rápida.
